ڕوم (ROM) : بریتیه‌ له‌ پارچه‌یه‌کی بچوک که‌ له‌سه‌ر بۆردی دایک جێگیرکراوه‌ ، وه‌ هه‌ر سێ پیتی (ROM) که‌ بۆ ناوی ئه‌م پێکهاته‌ ده‌نوسرێت
زانستی تۆماركردنی داتا لە سەر پەپکەکان زانستێكە كە بەهۆی تیشكی لەیزەری سور لە سی دی و دیڤیدیدا و شین لە جۆری دووەم لە دی ڤی دی كە بە (HD DVD)
رەنگە ئەو پرسیارەی لەسەرەوە كراوە بۆ زۆر لەو كەسانەی كە شارەزای ئەو بوارەن ئاسان بێت، بەڵام زۆر كەس هەیە كە تازە لە كۆمپیوتەر سەرەتایەتی و شارەزایەكی ئەوتۆی نییە، بۆیە بۆ ئەوان گەلێك سوودمەند دەبێت.
جار هەیە فایلێكی wordمان كردۆتەوە كە نیازمان هەیە ژمارەی بۆ دابنێین، بەڵام حەز ناكەین یەكەم لاپەڕە یان چەند لاپەڕەی پێشەوە، یان چەند لاپەڕەی ناوەند ژمارەیان هەبێ.
بۆ ئەم مەبەستە دەبێت لەسەرەتادا ئەگەر ئۆفێسەكەت 2003 بێت لەو لاپەڕەیەی كە نیازتە ئەو لاپەڕەیە ببێتە لاپەڕەی دەستپێك، كلیك بكەن، پاشان لەسەر مینۆی insert كلیك بكەن و دواتر فەرمانی Break لێبدەن و پاشان دوگمەی Page Break لێبدەن.
پاشان ok بكەن.
دواتر بچنە سەر مینۆی View و بەشی Header and footer و لەوێدا لە بەشی link to previous لە بەشی header and footer كلیك بكەن.
ئەوە دەبێتە هۆی ئەوەی كە ژمارە دانان هەر لەو لاپەڕەیە دەست پێبكات.
پاشان دەتوانی لاپەڕەكانی دیكە ژمارەیان بۆ دابنێی و لە بەشی format دا لە فەرمانی start at ئەو ژمارەی مەبەستتە بنووسی.
لە ئۆفێس 2007 و 2010شدا لە بەشی page layout كلیك لەسەر فەرمانی Break بكە، پاشان فەرمانی next page لێبدە.
دواتر بچۆ سەر فەرمانی insert و لە بەشی Header یان Footer هەڵبژێرە و پاشان فەرمانی Edit header یان Edit footer لێبدە.
لە دوگمەی رەنگ سەوزی Design دا بچۆ سەر بەشی Link to Previous كلیك بكە.
پاشان فەرمانەكانی دیكە وەك ئۆفێس 2003 ئەنجام بدە.
